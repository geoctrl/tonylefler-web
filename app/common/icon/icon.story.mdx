# Icon

SVG icons loaded from an optimized sprite sheet with automatic color inheritance.

## Import

```tsx
import { Icon } from "root";
```

## Basic Usage

<Example>

    <Icon name="star" />
  </div>
</Example>

```tsx

<Icon name="gear" />
```

## Icon System

Icons are sourced from an SVG sprite sheet for optimal performance:

- **Font Awesome** is the primary icon library
- Custom icons can be added by placing SVG files in the `icons/` directory
- Build process generates a sprite sheet and TypeScript types
- Icons are referenced by name (e.g., `name="house"`)
- TypeScript provides autocomplete for all available icon names

## Adding Custom Icons

To add custom icons to your project:

1. **Add SVG file**: Place your SVG icon in the `icons/` directory
2. **Restart dev server**: The build process automatically generates the sprite and types
3. **Use the icon**: Reference it by filename: `<Icon name="my-icon" />`

**Requirements:**
- SVG files must have a `viewBox="0 0 24 24"` attribute
- Keep icons simple and monochrome for best results with `fill-current`
- Font Awesome is the primary source, but any SVG icon can be added

## Accessibility

For decorative icons, use `aria-hidden`:

```tsx
<Icon name="star" aria-hidden="true" />
```

For semantic icons, add appropriate ARIA labels:

```tsx
<button aria-label="Close dialog">
  <Icon name="xmark" />
</button>
```
