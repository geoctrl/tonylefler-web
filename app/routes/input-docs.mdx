# Input

:: Single-line text field with some goodies built in.

This component is a wrapper around the native `input` element. It provides a
consistent look and feel across browsers and devices, and includes some
additional features like icons and labels.

<div className="max-w-xs">
  <Input placeholder="Type something..." />
</div>

```tsx
<Input placeholder="Type something..." />
```

---

## Props

Inherits all props from `HTMLElement`, including the following additional props:

- [formSize](#formsize)
- [iconLeft](#iconleft)
- [iconRight](#iconright)
- [label](#label)
- [onChange](#onchange)
- [value](#value)

### formSize

The size of the input. Defaults to `md`.

{

<div className="flex flex-wrap items-center gap-4">
  <Input formSize="sm" placeholder="sm" />
  <Input formSize="md" placeholder="md" />
  <Input formSize="lg" placeholder="lg" />
</div>
}

```tsx
<Input formSize="small" />
<Input formSize="medium" />
<Input formSize="large" />
```

### iconLeft

An icon to display on the left side of the input. Padding is automatically
added.

<div className="max-w-xs">
  <Input iconLeft="magnifying-glass" />
</div>

```tsx
<Input iconLeft="magnifying-glass" />
```

### iconRight

An icon to display on the right side of the input. Padding is automatically
added.

<div className="max-w-xs">
  <Input iconRight="percent" />
</div>

```tsx
<Input iconRight="percent" />
```

### label

The label for the input. Without an `id` prop, the label will not be clickable.

<div>
  <Input
    label="Label"
    id="my-input1"
    containerProps={{ className: "inline-block" }}
  />
  <Input
    label="Label"
    id="my-input2"
    containerProps={{ className: "inline-block" }}
  />
</div>

```tsx
<Input label="Label" id="my-input" />
```

### onChange

The change handler for the input.

<InputExample1 />

```tsx
import { useState } from "react";

function MyComponent() {
  const [val, setVal] = useState("");
  return <Input value={val} onChange={setVal} />;
}
```

### value

The value of the input.

<div className="max-w-xs">
  <Input value="Hello, World!" />
</div>

```tsx
<Input value="Hello, World!" />
```
