# Icon

:: Primitive SVG component using a spritesheet.

Set up with Vite using
[vite-plugin-icons-spritesheet](https://github.com/forge42dev/vite-plugin-icons-spritesheet)
with exported types.

  <Icon name="person-to-portal" />
</div>
