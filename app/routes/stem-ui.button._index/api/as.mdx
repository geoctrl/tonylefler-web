## as

```
as?: "button" | "a" | Link
```

The `as` prop allows you to render a button as a different HTML element or
component. Currently, the following values are supported: `"button"`, `"a"`, and
`Link`.

Defaults to `"button"`.

### `as="button"`

Render as a button element.

{<Button as="button">Hello, World!</Button>}

```tsx
<Button as="button">Hello, World!</Button>
```

### `as="a"`

Render as an anchor element.

{

<div className="not-prose">
  <Button as="a" href="https://remix.run/" target="_blank">
    Open remix.run
  </Button>
</div>
}

```tsx
<Button as="a" href="https://remix.run/" target="_blank">
  Open remix.run
</Button>
```

### `as={Link}`

Render as a react-router [Link](https://remix.run/docs/en/v2/api/remix#link)
component. Remix is used here, but you can use `react-router-dom` instead.
Element must be passed in.

{

<div className="not-prose">
  <Button as={Link} to="/">
    Go to home
  </Button>
</div>

}

```tsx
<Button as={Link} to="/">
  Go to home
</Button>
```
