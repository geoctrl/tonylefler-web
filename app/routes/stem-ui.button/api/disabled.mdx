import { Button } from "stem-ui";

### disabled

```
disabled?: boolean
```

As the name suggests, the `disabled` prop disables the button. The button
becomes non-interactive.

<div className="flex gap-2">
  <Button>Enabled</Button>
  <Button disabled>Disabled</Button>
</div>

```tsx
<Button>Enabled</Button>
<Button disabled>Disabled</Button>
```
