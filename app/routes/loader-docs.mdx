# Loader

Component that displays an indeterminate loading state.

## Import

```tsx
import { Loader } from "root";
```

## Basic Usage

<Loader />

```tsx
<Loader />
```

## Props

`<Loader>` accepts all [HTML div element attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/div) and the following:

<PropsTable props={loaderProps} />

### size

```
size?: "xs" | "sm" | "md" | "lg";
```

Size of the loader. Defaults to `md`.

{

<div className="flex gap-2">
  <Loader size="xs" />
  <Loader size="sm" />
  <Loader size="md" />
  <Loader size="lg" />
</div>
}

```tsx
<Loader size="xs" />
<Loader size="sm" />
<Loader size="md" />
<Loader size="lg" />
```

### intent

```
intent?:
 | "primary"
 | "primaryReverse"
 | "grey"
 | "greyReverse"
 | "onColor";
```

Color of the loader. Defaults to `primary`.

Color intent is purposefully built to work in a variety of situations.

`primary` and `grey` handle both light and dark backgrounds:

{

<div className="inline-flex gap-2 rounded-md border p-4 app-border">
  <Loader intent="primary" />
  <Loader intent="grey" />
</div>
}

```tsx
<Loader intent="primary" />
<Loader intent="grey" />
```

`primaryReverse` and `greyReverse` are the same as `primary` and `grey` but with
the theming reversed:

{

<div className="inline-flex gap-2 rounded-md bg-grey-900 p-4 dark:bg-grey-10">
  <Loader intent="primaryReverse" />
  <Loader intent="greyReverse" />
</div>
}

```tsx
<Loader intent="primaryReverse" />
<Loader intent="greyReverse" />
```

`onColor` is the same regardless of the theme - meant to be used on top of
elements with heavily saturated colors:

{

<div className="inline-flex gap-2 rounded-md bg-primary-500 p-4">
  <Loader intent="onColor" />
</div>
}

```tsx
<Loader intent="onColor" />
```
